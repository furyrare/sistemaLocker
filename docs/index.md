## 🎯 **Informações Rápidas**

### **Stack Completo**
- **Backend:** Node.js + Express + PostgreSQL + Prisma
- **Frontend:** React + Next.js + Tailwind CSS
- **Comunicação:** REST API

### **Portas Padrão**
- **Frontend:** 3000
- **Backend:** 4000
- **Database:** 5432

### **Estrutura de Pastas**
```
SistemaLocker/
├── backend/          # API e Database
├── frontend/         # Aplicação Web
└── docs/            # 📚 Documentação (aqui)
```

## 🔧 **Comandos Úteis**

### **Backend**
```bash
cd backend
npm install
npx prisma migrate dev
npm run dev
```

### **Frontend**
```bash
cd frontend
npm install
npm run dev
```

### **Database**
```bash
npx prisma studio
npx prisma generate
```
